# 🚀 PHP PDO CRUD

Projeto simples para estudo de **PHP + PDO**, implementando operações básicas de banco de dados.

## 📚 Tecnologias

- PHP
- MySQL
- PDO
- HTML

---

## 📂 Estrutura do Projeto

```
php_pdo
│
├── conexao.php
├── env.php
├── insert.php
├── read.php
├── update.php
└── delete.php
```

---

## ⚙️ Configuração

Configure seu banco no arquivo:

```
env.php
```

Exemplo:

```php
$host = "localhost";
$dbname = "meubanco";
$user = "root";
$pass = "";
```

---

## 🔗 Conexão com Banco

Arquivo responsável pela conexão:

```
conexao.php
```

Utilizando **PDO** para conexão segura com o banco de dados.

---

## ✏️ Operações CRUD

### ➕ Create

Arquivo:

```
insert.php
```

Responsável por inserir novos registros.

---

### 📖 Read

Arquivo:

```
read.php
```

Lista todos os usuários cadastrados.

---

### 🔄 Update

Arquivo:

```
update.php
```

Atualiza dados de usuários existentes.

---

### ❌ Delete

Arquivo:

```
delete.php
```

Remove registros do banco.

---

## 🧠 Objetivo do Projeto

Este projeto foi criado para praticar:

- Conexão com banco usando **PDO**
- **Prepared Statements**
- Operações **CRUD**
- Estrutura básica de backend em PHP
